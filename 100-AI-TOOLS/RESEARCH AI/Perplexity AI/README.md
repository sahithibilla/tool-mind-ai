# 🔎 Perplexity AI — Complete Guide

## 🧠 1. What is Perplexity AI?

Perplexity AI is an AI-powered search engine that provides direct answers to questions using information from reliable sources.

Unlike traditional search engines, Perplexity AI summarizes information and provides citations for the sources it uses.

It is widely used by students, researchers, and professionals for research, fact-checking, and quick information discovery.

### 🔎 AI Search Engine
