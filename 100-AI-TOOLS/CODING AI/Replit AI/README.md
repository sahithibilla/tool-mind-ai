# 💻 Replit AI — Complete Guide

## 🧠 1. What is Replit AI?

Replit AI is an AI-powered coding platform integrated into the Replit online development environment (IDE).  
It helps developers write, explain, debug, and deploy code directly in the browser without installing software.

The platform includes AI tools such as **Ghostwriter and Replit Agent**, which can generate code, analyze projects, and even build entire applications from natural language prompts. :contentReference[oaicite:1]{index=1}
