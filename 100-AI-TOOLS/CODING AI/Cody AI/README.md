# 💻 Cody AI — Complete Guide

## 🧠 1. What is Cody AI?

Cody AI is an AI-powered coding assistant developed by Sourcegraph that helps developers understand, write, and fix code more efficiently.

It is designed to work with large codebases and can analyze entire repositories to provide context-aware suggestions.

Cody AI is commonly used by development teams to navigate complex projects and improve productivity.

## 🎯 2. What Can Cody AI Do? (Core Functions)

### 💻 Code Generation
- Generate code snippets
- Create functions automatically
- Produce boilerplate code

### 🧠 Code Understanding
- Explain complex code
- Summarize files and functions
- Understand large repositories

### 🔎 Codebase Search
- Search across entire projects
- Locate functions and dependencies
- Navigate large codebases easily

### 🐞 Debugging Assistance
- Detect issues in code
- Suggest fixes
- Improve code quality
