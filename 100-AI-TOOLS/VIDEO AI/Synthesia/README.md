# 🎬 Synthesia — Complete Guide

## 🧠 1. What is Synthesia?

Synthesia is an AI video generation platform that allows users to create videos using AI avatars and text scripts.

Instead of recording yourself, you can type a script and Synthesia generates a video where an AI avatar speaks the text.

It is widely used for training videos, educational content, marketing videos, and corporate presentations.
