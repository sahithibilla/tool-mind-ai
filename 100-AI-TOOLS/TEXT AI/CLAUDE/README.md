# 🧠 Claude AI — Complete Guide

## 🧠 1. What is Claude?

Claude is an AI chatbot developed by Anthropic that can understand and generate human-like text.

It is designed to be:

- Safe  
- Helpful  
- Accurate  
- Ethical  

Claude is similar to ChatGPT but focuses more on long conversations and document analysis.
