# 🖌️ Uizard AI — Complete Guide

## 🧠 1. What is Uizard AI?

Uizard AI is an **AI-powered design platform** that helps users create **wireframes, mockups, prototypes, and UI designs** quickly.  
It’s built for **non-designers, product managers, startups, and developers** who want to turn ideas into interactive designs without needing advanced design skills.  

---

## 🎯 2. What Can Uizard AI Do? (Core Functions)

### 📐 Wireframe to Design
- Convert hand-drawn wireframes into digital mockups instantly  
- Transform sketches into interactive prototypes  

### 🎨 Design Assistant
- Generate UI designs from text prompts  
- Suggest layouts, color schemes, and components  

### 📱 App & Website Mockups
- Create mobile app screens, landing pages, and dashboards  
- Drag-and-drop editor for customization  

### 🤖 AI Features
- **Autodesigner:** Generate full app or website designs from a simple idea description  
- **Theme Generator:** Build custom themes from images or brand assets  
- **Copywriting AI:** Generate text for buttons, headlines, and descriptions  

### 🛠️ Collaboration
- Real-time team collaboration  
- Share prototypes with stakeholders  
- Export designs for developers
