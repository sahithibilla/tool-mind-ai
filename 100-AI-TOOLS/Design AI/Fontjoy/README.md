# 🔤 Fontjoy — Complete Guide

## 🧠 1. What is Fontjoy?

Fontjoy is a **free AI-powered font pairing tool** that helps designers, developers, and creators find harmonious font combinations.  
It uses **deep learning algorithms** to suggest pairings that balance contrast and similarity, making typography choices easier for branding, web design, and creative projects.

---

## 🎯 2. What Can Fontjoy Do? (Core Functions)

### ✨ Font Pairing
- Generate font combinations instantly  
- Balance similarity and contrast between fonts  
- Lock specific fonts and shuffle others  

### 🎨 Preview & Testing
- See font pairings applied to sample text  
- Adjust font sizes and weights  
- Test readability and aesthetics  

### 🔍 Font Exploration
- Browse Google Fonts library  
- Discover new typefaces  
- Save favorite pairings  

---

## ⭐ 3. Unique Features of Fontjoy

- **AI-driven font pairing** using deep learning  
- **Lock & shuffle system** for controlled experimentation  
- **Simple, clean interface** for quick testing  
- **Free and browser-based** (no installation required)
