# 🌐 Framer AI — Complete Guide

## 🧠 1. What is Framer AI?

Framer AI is an **AI-powered website design and publishing platform**.  
It allows users to generate **responsive layouts, content, and translations** directly from text prompts.  
Unlike traditional design tools, Framer outputs **live websites** — no developer handoff required.

---

## 🎯 2. What Can Framer AI Do? (Core Functions)

### 🖼️ AI Wireframer
- Generate responsive layouts from text prompts  
- Instantly create page structures for landing pages, portfolios, or dashboards  

### 🌍 AI Translate
- Translate entire websites into multiple languages  
- Maintain design consistency across translations  

### 🔌 AI Plugins
- Connect with external AI models (OpenAI, Anthropic, Gemini)  
- Generate images, rewrite text, or create alt text automatically  

### 🛠️ Workshop
- Build advanced components (cookie banners, tabs, effects) without coding  
- Extend functionality with reusable modules  

### 📊 CMS & SEO
- Integrated content management system  
- SEO optimization and analytics tracking  

### 🚀 Hosting & Publishing
- Publish directly from Framer  
- No need for external hosting or developer intervention
