# 🎨 Stable Diffusion — Complete Guide

## 🧠 1. What is Stable Diffusion?

Stable Diffusion is an open-source AI image generation model that creates images from text prompts.

Unlike many AI image generators, Stable Diffusion can be run locally on a personal computer, giving users full control over image generation.

It is widely used by artists, developers, and researchers for creating artwork and experimenting with generative AI.
