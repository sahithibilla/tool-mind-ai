# 🎨 NightCafe — Complete Guide

## 🧠 1. What is NightCafe?

NightCafe is an AI-powered image generation platform that allows users to create digital artwork using text prompts.

It provides multiple AI models for image creation, including Stable Diffusion and other generative AI techniques.

NightCafe is widely used by artists, designers, and hobbyists who want to experiment with AI-generated art.
